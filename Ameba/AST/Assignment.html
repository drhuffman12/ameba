    def transformed? #

    Indicates whether the node is a transformed assignment by the compiler. i.e.

    collection.each do |(a, b)|
  puts b
end

    is transformed to:

    collection.each do |__arg0|
  a = __arg0[0]
  b = __arg0[1]
  puts(b)
end
